À propos

GéniDev est né de la passion commune de quatre étudiants pour le code et la création web. Nous croyons que chaque ligne de code peut transformer des idées en solutions innovantes. Notre équipe s'efforce d'apprendre, de partager et de grandir ensemble, pour offrir le meilleur du développement web.

Compétences

HTML5
HTML5 est le langage de base du web : il structure le contenu des pages (titres, paragraphes, images, liens…) et permet l’affichage sur tous les navigateurs.
CSS3
CSS3 sert à styliser et mettre en forme les pages web : couleurs, mises en page, animations, adaptabilité mobile… Il rend le web beau et agréable à utiliser.
PHP
PHP est un langage de programmation côté serveur : il permet de créer des sites dynamiques, de gérer des bases de données et d’interagir avec les utilisateurs.
Java
Java est un langage polyvalent utilisé pour développer des applications robustes : sites web, logiciels, mobiles, objets connectés et bien plus encore.
